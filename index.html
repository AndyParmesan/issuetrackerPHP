XLSX Format Guide

Your XLSX file must include these columns in order:

ID | Particular | Module | Description | State | Status | Priority | Created By | Assigned To | Date Identified

Also supports extended format with Story Points, Sprint, Area Path columns after Priority.

The first row is treated as a header and skipped automatically.

e.g. MCC | Inventory | Login button not responding

💡 Assigned To is always imported as Unassigned — assign later via Edit.

Item states: New, Bug, Open, In Progress, As Designed, Enhancement, Needs Clarification, Monitoring, Closed

User Story states: Draft, For Review, Approved, For Development, In Development, For Testing, QA Failed, For UAT, Ready for Deployment, Deployed, Closed

Valid priorities: 1-Urgent, 2-Critical, 3-High, 4-Medium, 5-Low
